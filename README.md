# MARSS-for-crabs

R code to run a MARSS (multivariate autoregressive state space model) model integrating commercial and independent survey data for blue swimmer crabs. This code accompanies a manuscript Smith et al (in progress).

The data to run this is not shared here but can be requested from the main author: james.a.smith@dpi.nsw.gov.au

The main code is in BSC MARSS TOP SCRIPT.R file, which calls some functions in the FUNCTIONS.R file.
